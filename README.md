đây là dự án mini sử dụng js, html, css để làm app dự báo thời tiết và call api từ openweathermap
vào web https://home.openweathermap.org/users/confirmation đăng kí tài khoản và xác nhận email để nhận key api về thời tiết các địa điểm trên toàn thế giới
chờ đợi api được kích hoạt trong vòng một vài tiếng hoặc 1 vài ngày sau đó sử dụng, lưu ý , key API có thể hết hạn sau 1 khoảng thời gian.
